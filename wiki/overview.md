# Wiki Overview

LLM Wiki separates raw captures from durable knowledge.

- `raw/inbox/`: quick captures and file drops
- `raw/sources/`: accepted source records
- `wiki/sources/`: source pages derived from accepted records
- `wiki/concepts/`, `wiki/entities/`, `wiki/projects/`: structured knowledge
- `wiki/answers/` and `wiki/syntheses/`: reusable agent outputs

The public repository includes the implementation and folder structure, but not
private captured content or generated personal notes.
